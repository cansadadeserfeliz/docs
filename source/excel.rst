================================================
Какой content-type передавать для файлов Excel?
================================================

Для файлов BIFF .xls: ``application/vnd.ms-excel``

Для Excel2007 выше .xlsx: ``application/vnd.openxmlformats-officedocument.spreadsheetml.sheet``
